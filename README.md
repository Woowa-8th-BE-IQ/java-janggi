## 구현할 기능 목록

### 열 좌표
- [X] 1~9까지의 숫자로 열 좌표를 생성하면 올바른 열 좌표를 가지고 있다.
- [X] 범위 밖의 숫자로 열 좌표를 생성하면 예외가 발생한다.

### 행 좌표
- [X] 1~10까지의 숫자로 행 좌표를 생성하면 올바른 행 좌표를 가지고 있다.
- [X] 범위 밖의 숫자로 행 좌표를 생성하면 예외가 발생한다.

### 위치
- [X] 좌표를 입력받아 위치를 생성한다.
- [X] 존재하지 않는 좌표로 위치를 생성하면 예외가 발생한다.
- [X] 좌표가 같은 위치는 동등하다.

### 기물
- [X] 팀이 맞는지 판정한다.
- [X] 자신의 이름을 반환한다.
- [X] 차, 마, 상, 장, 사, 포, 졸, 빈 구현한다.

### 보드생성
- [X] 보드에 모든 기물을 배치한다.
- [X] 보드 마상 스왑 기능 구현한다.
- [X] 초기화된 보드를 반환한다.

### 보드
- [X] 보드 상태를 방어적 복사하여 반환한다.


## 이동 규칙 구현
Board.move(from, to)
→ piece.getPath(from, to) : List<Position> // 경로
→ 경로의 Piece들 추출
→ piece.canMove(List<Piece> /* 경로 */, toPiece) : void (예외)
→ 이동 실행

### 장 / 사
- [X] 직선 한 칸을 이동시키면 경로를 반환한다.
- [X] 대각선 이동시키면 예외가 발생한다.
- [X] 직선 한 칸보다 많이 이동시키면 예외를 발생한다.
- [X] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.
- [X] 이동할 위치에 다른 팀이 있으면 참을 반환한다.

### 차
- [X] 한 방향으로만 이동시키면 경로를 반환한다.
- [X] 여러 방향으로 이동하면 예외가 발생한다.
- [X] 경로에 존재하는 기물 중 빈 기물이 아닌 기물이 있으면 예외가 발생한다.
- [X] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.

### 포
- [X] 여러 방향으로 이동하면 예외가 발생한다.
- [X] 경로에 존재하는 기물이 1개가 아니면 예외가 발생한다.
- [X] 경로에 존재하는 기물이 포면 예외가 발생한다.
- [X] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.
- [X] 이동할 위치에 적의 포가 있으면 예외가 발생한다.
- [X] 한 방향으로만 이동시키면 경로를 반환한다.

### 마
- [X] 직선으로 먼저 한 칸, 직선 방향의 대각선으로 한 칸 이동시키면 경로를 반환한다.
- [X] 그 외의 방향으로 이동시키면 예외가 발생한다.
- [ ] 경로에 존재하는 기물 중 빈 기물이 아닌 기물이 있으면 예외가 발생한다.
- [ ] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.
### 상
- [ ] 직선으로 먼저 한 칸, 직선 방향의 대각선으로 연속 두 칸 이동시키면 경로를 반환한다.
- [ ] 그 외의 방향으로 이동시키면 예외가 발생한다.
- [ ] 경로에 존재하는 기물 중 빈 기물이 아닌 기물이 있으면 예외가 발생한다.
- [ ] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.

### 빈
- [ ] 경로를 요청받으면 예외를 발생시킨다.

### 쫄
- [ ] 직선 한 칸을 이동시키면 경로를 반환한다.
- [ ] 대각선 이동시키면 예외가 발생한다.
- [ ] 팀 기준 뒤로 이동하면 예외가 발생한다.
- [ ] 직선 한 칸보다 많이 이동시키면 예외를 발생한다.
- [ ] 이동할 위치에 같은 팀이 있으면 예외가 발생한다.
