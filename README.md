## 구현할 기능 목록

### 열 좌표
- [X] 1~9까지의 숫자로 열 좌표를 생성하면 올바른 열 좌표를 가지고 있다.
- [X] 범위 밖의 숫자로 열 좌표를 생성하면 예외가 발생한다.

### 행 좌표
- [X] 1~10까지의 숫자로 행 좌표를 생성하면 올바른 행 좌표를 가지고 있다.
- [X] 범위 밖의 숫자로 행 좌표를 생성하면 예외가 발생한다.

### 위치
- [X] 좌표를 입력받아 위치를 생성한다.
- [X] 존재하지 않는 좌표로 위치를 생성하면 예외가 발생한다.
- [X] 좌표가 같은 위치는 동등하다.

### 기물
- [X] 팀이 맞는지 판정한다.
- [X] 자신의 이름을 반환한다.
- [X] 차, 마, 상, 장, 사, 포, 졸, 빈 구현한다.

### 보드생성
- [X] 보드에 모든 기물을 배치한다.
- [X] 보드 마상 스왑 기능 구현한다.
- [X] 초기화된 보드를 반환한다.

### 보드
- [X] 보드 상태를 방어적 복사하여 반환한다.

### 이동 규칙 구현 진행
1. 장이 이동 불가능한 거리면 예외 발생
2. 사가 이동 불가능한 거리면 예외 발생
3. 차 이동 규칙
4. 포 이동 규칙
5. 마 이동 규칙
6. 상 이동 규칙
7. 졸 이동 규칙
